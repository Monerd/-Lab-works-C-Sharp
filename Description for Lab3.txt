3-ья лабораторная основана на второй, за исключением добавление парсинга XML и JSON файлов.
При запуске создаются три папки по пути:
1. D:\Lab3
2. D:\Lab3\SourceDirectory
3. D:\Lab3\TargetDirectory
соответсвенно ( данные пути для создания берутся из конфигурации JSON файла, ибо в изначальном
коде в метод класса Parse передаётся именно этот файл ).
Пути папок можно изменить в конфигурационных файлах ( Configuration.xml | Configuration.json ).
Все остальные действия и фичи аналогичны 2-ой лабораторной работе.
Помимо всего прочего, в 3-ей лабораторной классы разбиты по файлам, создан интерфейс IParser,
реализующий XMLParser и JSONParser, и изменена архивация файлов.
Создана схема .xsd для подтверждения .xml файла ( метод Affirm в классе XMLParser ).
Для запуска потребуется скачать некоторые пакеты через "управление пакетами NuGet".
Изменено название сервиса: Service1 -> FileWatcherService
