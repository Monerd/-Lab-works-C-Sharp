4-ая лабораторная работа заимствует некоторую конфигурацию из 3-ей лабораторной работы для
работы с JSON и XML файлами, однако они очень сильно отличаются и в каком-то смысле очень сильно изменены.

ConfigurationManager - проект с классами для считывания данных из XML & JSON файлов
DataAccessLayer - проект ( пройслойка | n-layer ), содержащий классы для работы непосредственно с БД
ServiceLayer - проект, содержащий классы для представления пользователю службы данных из БД
Models - проект, содержащий модели для обработки данных нашей службой
DataBaseService - непосредственно наша служба
FileWatcherService - служба для мониторинга FTP-сервера, в нашем случае - двух папок ( SourceDirectory &
TargetDirectory ), создаваемых после запуска службы ( подробнее в "Description for Lab3" )
SQLProcedure - папка, в которой находится хранимая процедура ( сама хранимая процедура описана в app.config
нашей службы )

С большего все проекты расписаны. Зависимости, сервисы для создания XML и отправки создавшихся файлов имеются,
что показывается в видео-описании.
